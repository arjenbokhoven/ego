ego - A CSS Framework

ego is a CSS framework built while coping with boredom.

Container class

Encapsulating all the content is a container class in three different breakpoints.

Breakpoints

small medium large
720px 1000px 1200px

Flexbox grid

The flex-l grid items have breakpoints of 1000 pixels of their parent container. The flex-s grid items have breakpoints of 720 pixels of their parent container. This mobile-first approach makes all grid items stack on small screens.
