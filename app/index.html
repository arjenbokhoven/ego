ego - A CSS Framework

ego is a CSS framework built while coping with boredom.

The "content" & "section" classes

Encapsulating all the content is a "container" class that keeps content at a width of 768 pixels, and then shrinks down to 100% on any device with a lower screen resolution.

The "section" class has a nice openness with a bottom margin set to 6rem.

Flexbox grid

Using rows and columns with modifiers, it's easy to make interesting layouts. Who needs 12 columns?.
